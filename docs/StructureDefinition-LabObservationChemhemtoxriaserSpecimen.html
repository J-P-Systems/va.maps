Resource Profile: Lab Observation: Chem, hem, tox, ria, ser Specimen

Official URL: http://va.gov/fhir/StructureDefinition/LabObservationChemhemtoxriaserSpecimen Version: 1.5.30
Active as of 2025-05-25 Computable Name: LabObservationChemhemtoxriaserSpecimen

This StructureDefinition contains the maps for VistA file CHEM, HEM, TOX, RIA, SER, etc. (63.04) to Specimen.

​See also the FHIR Shorthand.

Use Case Overview

Usage:

  • Refer to this Resource Profile: Lab Observation: Chem, hem, tox, ria, ser Observation
  • CapabilityStatements using this Resource Profile: Lighthouse Clinical Health API and Lighthouse Patient Health API
