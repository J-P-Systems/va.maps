Resource Profile: Procedure: Surgery Location[OR]

Official URL: http://va.gov/fhir/StructureDefinition/ProcedureSurgeryLocationOR Version: 1.4.10
Draft as of 2025-04-18 Computable Name: ProcedureSurgeryLocationOR

This StructureDefinition contains the maps for VistA file OPERATING ROOM (131.7) to us-core-location.

​See also the FHIR Shorthand.

Algorithm for populating Procedure.performed

  • If there is only one date or time, use that value as the performedDateTime.
  • If there is one date and one time, use the time as the performedDateTime.
  • If there is more than one time, put the earliest in performedPeriod.start and the latest in performedPeriod.end, wherever they come from.

The following VistA SURGERY (130) file fields contain dates:

  • DATE OF OPERATION (.09)
  • TIME PROCEDURE BEGAN (121)
  • TIME PROCEDURE ENDED (122)
  • TIME OPERATION BEGAN (.22)
  • TIME OPERATION ENDS (.23)

Use Case Overview

Usage:

  • Refer to this Resource Profile: Procedure: Surgery Procedure
  • CapabilityStatements using this Resource Profile: Lighthouse Patient Health API
