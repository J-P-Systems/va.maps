Resource Profile: Procedure: Surgery Location[OR]

Official URL: http://va.gov/fhir/StructureDefinition/ProcedureSurgeryLocationOR Version: 0.69.438
Active as of 2024-12-06 Computable Name: ProcedureSurgeryLocationOR

This StructureDefinition contains the maps for VistA file OPERATING ROOM (131.7) to us-core-location

Algorithm for populating Procedure.performed[x]:
  1. If there is only one date or time, use that value as the performedDateTime.
  2. If there is one date and one time, use the time as the performedDateTime.
  3. If there is more than one time, put the earliest in performedPeriod.start and the latest in performedPeriod.end, wherever they come from.
The following VistA SURGERY (130) file fields contain dates:
DATE OF OPERATION (.09), TIME PROCEDURE BEGAN (121), TIME PROCEDURE ENDED (122), TIME OPERATION BEGAN (.22), TIME OPERATION ENDS (.23)

Usage:

  • Refer to this Resource Profile: Procedure: Surgery Procedure
